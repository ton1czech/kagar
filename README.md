## üóíÔ∏è **INSTALLATION**

### local installation:

1. clone the repo

```
git clone https://github.com/ton1czech/kagar
```

2. cd into `kagar`

```
cd kagar
```

3. install dependencies

```
go mod tidy
```

4. run the app

```
go run kagar.go

// or

go build
./kagar
```
